# private_chat

A private, self-destructing chat room for 2 people. Create ephemeral rooms that auto-delete after a set time, with real-time messaging.

**Live:** [realtime.sachi.dev](https://realtime.sachi.dev)

## Features

- Real-time messaging
- Self-destructing rooms with countdown timer
- Anonymous usernames
- Manual room destruction
- Dark/light mode
